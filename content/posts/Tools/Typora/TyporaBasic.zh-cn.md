## 快捷键

以下仅记录常用快捷键，详细完整版参考[官方快捷键文档](https://support.typora.io/Shortcut-Keys/)。

| 功能         | 按键              |
| ------------ | ----------------- |
| 查找         | `Ctrl + F`        |
| 查找和替换   | `Ctrl + H`        |
| 偏好设置     | `Ctrl + ,`        |
| 选中当前行   | `Ctrl + L`        |
| 插入正文     | `Ctrl + 0`        |
| 全屏         | `F11`             |
| 插入链接     | `Ctrl + K`        |
| 增加标题级数 | `Ctrl + =`        |
| 降低标题级数 | `Ctrl + -`        |
| 删除线       | `Alt + shift + 5` |
| 下划线       | `Ctrl+U`          |
