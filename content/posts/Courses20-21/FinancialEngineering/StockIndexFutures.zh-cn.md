## 携带成本模型

携带成本模型(cost of carry model)可以利用无套利方法与复制资产方法两种方法来推导。

### 无套利方法

*无套利假设*是指市场上不存在这样的机会，即不用承担任何风险同时还能获得正收益。这一看似简单的无套利假设其实背后蕴含着很多市场机制的前提条件：

1. 比如*充分竞争*，如果存在垄断行为，那么套利空间是可以长期存在的；
2. 比如*信息对称*，如果市场一方拥有绝对的信息优势，那么这些投资者就有无风险套利的机会；
3. 再比如*自由交易*，如果存在交易限制，那么即便出现了套利机会市场投资者也无法捕捉。

这些条件的不满足，都会影响无套利假设的合理性。从而影响模型的定价结果。

|      |                                                     | 现金流出 | 现金流入                                    |
| ---- | --------------------------------------------------- | -------- | ------------------------------------------- |
|      | 操作(T=0)                                           | T=0      | T=1                                         |
| 现货 | 通过债券融资$S_0$并买入与指数对应的一篮子成分股股票 | 0        | $S_{1}+D_{0,1}-S_{0}\left(1+r_{0,1}\right)$ |
| 期货 | 买入合约面值为$S_0$的股指期货                       | 0        | $S_1-F_{0,1}$                               |

根据无套利假设，$S_{1}+D_{0,1}-S_{0}\left(1+r_{0,1}\right)=S_1-F_{0,1}$，从而有期货定价公式
$$
F_{0,1}=S_{0}\left(1+r_{0,1}\right)-D_{0,1}
$$

### 复制资产方法

这里的假设条件是金融市场是完备的。在一个*完备市场(complete market)*里面，任何金融资产的未来回报结构都可以由现有的其他资产所形成的组合复制出来。

**完备市场是无套利充分非必要条件**：完备市场一定是无套利的，无套利的市场不一定完备。

|      |                                                     | 现金流  | 现金流                                  |
| ---- | --------------------------------------------------- | ------- | --------------------------------------- |
|      | 操作(T=0)                                           | T=0     | T=1                                     |
| 现货 | 通过债券融资金额B并买入与指数对应的一篮子成分股股票 | $S_0-B$ | $S_{1}+D_{0,1}-B\left(1+r_{0,1}\right)$ |
| 期货 | 买入股指期货                                        | 0       | $S_1-F_{0,1}$                           |

现货与期货的期初现金流相等，则未来的现金流页必然相等：当$ S_{0}-B=0 $时，$S_{1}+D_{0,1}-B\left(1+r_{0,1}\right)=S_{1}-F_{0,1}$，从而求得
$$
F_{0,1}=S_{0}\left(1+r_{0,1}\right)-D_{0,1}
$$
**仅从公式计算来看，期货的价格决定于现货的现价与无风险利率，与现货的未来价格/未来收益率无关**，期货价格确实体现了现货价格的增长，但增长率却不是现货的预期增长率，而是扣除红利率之后的无风险利率！

根据携带成本模型，一般远月合约价格>近月合约价格>现货价格，这与现实相符。

### 携带成本模型的局限

假设现货价格是外生的，忽略了期货与现货市场之间的*动态互动*；**事实上，现货价格已经反映了部分未来现货价格的预期，相应的期货价格也反映了未来价格的预期**。

实证研究也发现，基于携带成本模型计算的价格与市场价格存在**系统性偏差**。

定价误差，在期货刚刚推出的时候定价误差比较大，一度时间后，逐步缩减到交易成本以内。
$$
定价误差 = \left(\boldsymbol{F}^{\text {市场 }}-\boldsymbol{F}^{\text {公允 }}\right) / \boldsymbol{F}^{\text {市场 }}
$$

- 定价误差具有自相关性
- 定价误差与到期日、利率水平及市场流动性等因素有关

#### 市场波动性影响定价

Chen et al.(1997)将投资者分为三大类

1. 对冲者：现货多头+股指期货空头
2. 现货替代者：现货多头+股指期货多头
    1. 现货头寸可能要求，例如股票型基金需要持有一定比例的股票头寸
    2. 期货多头用于节约建仓成本
3. 观望者：既没有现货也没有期货

**当波动性增加时**，对冲压力加大，对冲者的空头头寸增加，现货替代者的多头头寸减少，**股指期货的空头力量变大，容易出现负基差**。

韩乾和梁巨方(2017)：市场尾部风险溢价是股市异常波动期间股指期货价格的重要影响因素。

## 股指期货的基差

### 基差影响对冲组合的收益率

基差是期货价格与现货价格的差，即

$$
B = F-S = S*(r-d)
$$
注意，日常使用中

- 金融期货的基差一般定义为$F-S$

- 而商品期货的基差一般定义为$S-F$

- 还有的将基差定义为$log F-log S$，反映了基差占现货价格的比例

    其中，

- $B<0$，称为*负基差/贴水*

- $B>0$，称为*正基差/升水*

对冲组合的收益率受到基差的影响

- 例一：期货持有至到期收益
- 例二：期货持有未到期的收益

|                                  | 现金流出 | 现金流入(例一)                       | 现金流入(例二)                                 |
| -------------------------------- | -------- | ------------------------------------ | ---------------------------------------------- |
| 操作(T=0)                        | T=0      | T=1                                  | T=0.5                                          |
| 买入并持有与指数对应的一篮子股票 | $S_0$    | $S_{1}+D_{0,1}$                      | $S_{0.5}+D_{0,0.5}$                            |
| 卖空股指期货                     | 0        | $F_{0}-S_1$                          | $F_{0}-F_{0.5}$                                |
| 对冲组合                         | $S_0$    | $F_{0}-D_{0,1}$                      | $F_{0}-F_{0.5}+S_{0.5}+D_{0,0.5}$              |
| 收益率                           |          | $\left(B_{0}+D_{0,1}\right) / S_{0}$ | $\left(B_{0}-B_{0.5}+D_{0,0.5}\right) / S_{0}$ |

从组合收益率可以看出，期货的负基差会降低对冲组合收益率，也即提高了期货避险的成本。

### 我国股指期货市场长期负基差

**我国股指期货市场长期负基差**：自2015年6月20日股价异常波动以来，股指期货基差大幅贴水，这一现象至今仍存在。

![](https://cdn.jsdelivr.net/gh/henrywu97/FigBed@master/Figs/20210506162723.png)

### 西方市场出现负基差的原因不能解释我国的负基差

西方市场过去出现的负基差可能的解释：

#### 反向套利的成本允许负基差存在

假设根据携带成本模型计算的期货价格为$F^{*}$,

1. 当$F>F^{*}$，采用*正向套利*获取无风险利润：卖空股指期货，买入一篮子股票
2. 当$F<F^{*}$，采用*负向套利/ 反向套利*获取无风险利润：买入股指期货，卖空一篮子股票。

在我国，**正向套利相较于负向套利需要通过融券卖出现货的做法，限制较少，成本较低**。因此在正基差走阔时，市场上总会存在套利者形成促 使正基差回归的力量。这也解释了历史上正基差绝对值相较于负基差绝对值要更小的原因。

反向套利需要通过融券卖出现货，在我国的市场环境下，市场的**融券机制不够完善，融券标的少，融券规模小**。

通过*提前购买标的股票建仓*的方式也可以实现卖出套利，但是建仓成本很高。

反向的套利的高成本允许负基差在一定范围内存在。

但是根据无套利原则，考虑构建负向套利组合的成本和费用，自2015年股灾以来，**股指期货实际结算价格较大程度地低于其无套利价格区间下限**。实际基差相对于理论基差的偏差均值甚至达到了5倍以上。

![](https://cdn.jsdelivr.net/gh/henrywu97/FigBed@master/Figs/20210507170402.png)

具体无套利区间计算方法参见[中国股指期货基差分析](http://www.ssf.gov.cn/yjypz/201608/P020160831620342964930.pdf)。

因此**反向套利成本仍不能完全解释我国股指期货的负基差**。

#### 集中分红

根据携带成本模型$B = S*(r-d)$，当分红收益率要高于无风险收益率时，理论上期货价格会出现贴水。

沪深 300 指数成分 股上市公司的分红较为集中，通常每年分红一次，时间往往是自然月的 5、6、7 月。

2010年以来沪深300指数成分股按月的分红收益率。

![](https://cdn.jsdelivr.net/gh/henrywu97/FigBed@master/Figs/20210507165106.png)

以2015年股灾为分界线：

1. 自上市以来*截至2015 年股灾之前*，沪深300股指期货大部分时间都处于升水。而期货出现贴水往往具有一定的季节性，大部分出现在5、6、7月三个月，随后回到升水。5、6、7 月的交易日中，即便期货不出现贴水，正基差也较小。利用携带成本模型计算的5、6、7月的**理论基差与实际基差的偏差较小，贴水可以由A股的集中分红来解释**。
2. 自*2015年股灾以来*，我们可以看到，沪深300股指期货**长时间处于深幅贴水**，随着7月份分红实施结束，股指期货的贴水也并没有消失。这表明：**当前股指期货的贴水已不能由分红收益率高于无风险收益率解释**。

#### 非同步交易

市场暴跌时现货指令堵塞/跌停/停牌，期货正常交易下跌，负基差将持续扩大;

- 此时套利者将买入期货，卖出现货，现货堵塞程度将更加严重，从而加剧负基差，形成*恶性循环*

2015年股灾时期，出现了严重的跌停。

![](https://cdn.jsdelivr.net/gh/henrywu97/FigBed@master/Figs/20210506172404.png)

停牌公司数量也有显著增加：

![](https://cdn.jsdelivr.net/gh/henrywu97/FigBed@master/Figs/20210506172359.png)

**但跌停与停牌的非连续性并不能解释持续性的负基差**。

#### 波动性与流动性

波动性：Chen et al.(1997)，前文已有解释

流动性：流动性较差的情况下，套利成本提升，允许负基差的存在

波动性与流动性也不能很好地解释我国负基差。

### 尾部风险溢价对股灾期间负基差具有解释力

尾部风硷溢价(韩乾和梁巨方，2017)：尾部风险溢价对于股指期货负基差具有很强地解释力。

理论解释：市场担心股市的尾部风险，通过卖空股指期货进行对冲，从而带来负基差的压力。

下图中，蓝色曲线代表尾部风险溢价，2015股灾发生后，尾部风险溢价显著提升。

**尾部风险能解释股灾期间的负基差**，但是不能解释股灾之后的负基差。

## 股指期货的套期保值

由于期货与现货之间的线性相关关系，可以用股指期货对冲：

- 单只股票，一般高$Beta$股票才需要对冲，例如茅台；
- 一个股票组合，主要的对冲需求来源；
- 一篮子指数成分股股票

股指期货套期保值与商品期货套期保值的不同

- 商品期货的套保成本可以用实体经济利润来负担
- 金融期货的套保成本则需要额外支付

### 套期保值的成本

根据携带成本模型，如果对一篮子指数成分股股票进行套保，则
$$
收益率 = \left(B_{0}+D_{0,1}\right) / S_{0} = r_f
$$
如果不套保，回报率为指数预期增长率$\mu$。

因此套保者向其他交易者“支付”了风险溢价$\mu-r$。卖出套保平均而言是要支付风险溢价的。

![](https://cdn.jsdelivr.net/gh/henrywu97/FigBed@master/Figs/20210603210819.png)

受到市场上其他因素影响，携带成本模型常常不成立，此时基差将成为套保成本的主要影响因素。

如何看待套期保值的贏亏？

- 商品期货并没有贏亏的绝对概念
- 股指期货也是一样
- 但是套期保值都是有成本的，不能把成本看做是亏损

阿尔法策略的超额收益如果不能覆盖套期保值成本，那么策略是亏的。

能够在市场上生存下来的机构其实时刻都在套保（包括盘中），不留一点风险敞口(阿尔法策略除外)。

当投机不足的时候，多头力量缺乏时，期货价格较低，套保者的整体利润更低。

- 投机（好的市场流动性）有利于降低套保成本。
- 降低市场波动性有利于降低套保成本。
- 套保成本越低，越有利于稳定持有现货。

![](https://cdn.jsdelivr.net/gh/henrywu97/FigBed@master/Figs/20210603211345.png)

中国企业的现金流普遍紧张，当套保存在亏损时，常常被强行平仓。

### 最优对冲比率

股指期货的最优对冲比率公式为
$$
h=\rho_{i, m} \frac{\sigma_{i}}{\sigma_{m}}
$$
根据在CAPM模型中，
$$
\beta=\frac{\operatorname{Cov}\left(r_{i}, r_{m}\right)}{\sigma_{m}^{2}}
$$
进一步有
$$
h = \frac{\operatorname{Cov}\left(r_{i}, r_{m}\right)}{\sigma_{i} \sigma_{m}} \frac{\sigma_{i}}{\sigma_{m}} = \frac{\operatorname{Cov}\left(r_{i}, r_{m}\right)}{\sigma_{m}^{2}}=\beta
$$
因此可以用Beta作为股指期货对冲比率的替代指标。

## 股指期货的价格发现

携带成本模型的最大局限在忽略了期现之间的动态互动过程，并假设现货价格是外生的。

信息融入现货和期货价格的动态过程就是市场的价格发现过程。

一般来讲，期货价格会更快地反应新的信息冲击

- 期货市场流动性更好
- 期货市场交易成本更低
- 期货市场具有高杠杆特性

但并不总是如此！

股指期货的价格发现功能的三个衡量指标的时间序列

- 价格发现的衡量方法在学术界仍处于探索阶段
- 从时间序列来看，**期货非常好的发挥了价格发现的功能**

![](https://cdn.jsdelivr.net/gh/henrywu97/FigBed@master/Figs/20210507170819.png)

上图的信息份额很可能**高估了股指期货的价格发现功能**：

- *更新频率*：我国股票指数的行情更新频率是5秒一次，而股指期货的行情更新频率是0.5秒一次，这意味着如果我们采取1秒频率的数据来测算信息份额的话，此时股指期货的价格已经发生变化，但股票指数水平仍然保持不变，这就导致了股指期货价格发现功能由于技术原因得到高估。事实上，根据韩乾和赵诚志(2019)的测算，**由于股票行情更新频率不同步所导致股指期货的价格发现功能平均高估6.71%左右，期货的领先时间被高估40%以上**。
- *忽略了期权市场的价格发现功能*：《上证50指数现货、期货和期权市场之间的价格发现》指出，全样本期内，股指期货市场的价格发现份额约为 34%，期权市场约为 41%；如果只考虑ETF现货市场和期权市场，那么期权市场的信息份额占到 67.76%；而如果只考虑ETF现货市场和期货市场，那么期货市场的信息份额也占到了 67.42%。这说明，此前的研究在未考虑期权市场的情况下将**一部分期权市场该有的信息份额被‘错误地’归到期货市场**，从而高估了期货市场的价格发现功能。详细参见[关于50ETF期权，学者们有哪些研究成果的第三篇](https://mp.weixin.qq.com/s/LrEdwnr7ryY4y6EFbiJDcQ)。

### 定价权

股指期货对于股指是否有定价权？

每次股市大波动之前，总是期货价格先下跌，这是不是意味着期货导致股票价格下跌？

- 股市下跌真正的原因是外部的信息冲击，只不过反应的速度不同

对比商品期货，**商品市场确实存在期货对现货进行定价的逻辑**：

- 大部分商品现货没有全国统一的市场，各地方现货价格不相同
- 商品期货价格可以作为现货市场的重要参考
- *点价机制*：现货商签合约时习惯以期货价格加点的模式进行

例如：铁矿石期货推出的初衷是获取铁矿石的国际定价权。但目前我国铁矿石市场受到操纵，暴涨暴跌。

与商品期货不同，在股指期货市场上

- 股票市场是全国统一的，且股票市场的流动性足够好，自身具有定价权
- 事实上，很难想象股指期货能 定价指数
    - 指数由各个成分股计算得到
- 股市走势自有其规律，受到经济基本面、政策因素、投资者行为等的影响，跟股指期货无关

### 助涨助跌

有以下观点认为，趋势交易就会助涨助跌：

> 我不知道为什么“松绑”股指期货就会提振信心，因为**股指期货就是助涨助跌**的，如果你不是用来做风险控制，来做对冲，它就是助涨助跌，涨的时候如果趋势是涨的，谁去买跌呢？他一定都去买涨，买涨那不就助涨吗；跌的时侯明明趋势是跌，谁去买涨呢？如果出于做投机套利股指期货的投机行为的话，它绝对是助涨助跌。
>
> 
>
> “股指股指期货价格通过套期保值和期现套利，向现货指数传递涨跌。**当股票市场整体下跌时，投资者就会卖出股指期货套保，从而使股指期货下跌速度快于股票指数**。当两者之间价差超过无套利区间时，期现套利盘就会买入股指期货，并同时卖出股票，而股票的抛单又引发股票指数加速下跌。这种期现市场的价格传递，就是股指期货价格助跌股票指数的根本原因"
>
> 
>
> “反之，股指期货也会通过价格传递来助涨现货指数“
>
> 
>
> “程序化交易又将这种价格传递速度显著提高”

对以上观点的批判：

- 套保并非仅在下跌时进行套保，而是在下跌前提前建仓；
- 套保也不一定使股指期货下跌速度快于股票指数
- 期货套利盘卖出股票也并没有那么容易，提前建仓不现实，而融券成本又较高。

#### 解读

股票市场的容量远远大于股指期货市场

- 60万亿的市场 v.s. 600亿-1500亿的持仓
- 成交金额：6000亿-1万亿 v.s. 700亿-2000亿
- 体量更小，面对同样的信息，波动就会更大。

股指期货市场上的多空博弈非常激烈

- 股票市场如果实现T+0也会增加日内波动

股指期货的交易限制影响了流动性

- 2015年故事异常波动期间，交易所对股指期货交易进行了一系列限制包，括开仓头寸、保证金、手续费等

### 操纵期指从现货获利

例如以下观点：

> 那么，机构如何通过期指获利呢？很简单，首先在期指上做空，然后在A股卖出证券等蓝筹股。机构通过卖出A股导致指数下跌，确保了自己在期指的盈利，然后尾盘再接回A股，不仅没有丢失筹码还在期指获得盈利，散户就只能眼睁睁看着下跌，无法应对。
>
> 
>
> 这不是意淫，是切实可操作的，我们可以明显感觉早盘的大金融一度是很强的，但尾盘一小时内却暴跌，个中操作手法慢慢体会吧。
>
> 
>
> 如果没有期指，由于早盘做多力量的买入，其实，今天是不会如此暴跌的。但期指的放开加剧了尾盘的暴跌，使其成为机构鱼肉散户的工具。

批判：

- 期指上做空建仓需要比较大的量，可能对市场造成价格冲击，建仓成本较高；
- 打压A股需要的股票仓位很大，对应的资金量非常大，股票建仓的成本也很高；
    - 测算打压至跌停所需要的资金量：利用LOB(limit order book)的报单，加总至跌停价格所需要的成交量与资金的乘积。
- 另外利用融券来作空股票也不太可能：做空股票会使券商亏损(券商是融券的对手方，到期要还股票给券商)，因此券商不会提供过多的券源。
